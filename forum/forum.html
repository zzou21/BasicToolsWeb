Forum

Help Basic Tools Improve!

Basic Tools is designed to be an app that serves the public, so I welcome public suggestions and recommendations about what new tools or features that you think I should include in a future edition of Basic Tools. Simply drop your recommendation in the forum below or see what others have already recommended. You can also react to an existing suggestion to show your support for it!
You do not need an account to make a post. You just need to put in your email before you post so that we can verify that you are not a bot.

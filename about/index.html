About Basic Tools

Current version: 1.3.36

Basic Tools currently has x tools, where x is the middle value (minor version #) in the version number, e.g., version 0.x.00.

An open-source and minimum-computing suite of everyday computer utilities designed to run locally on your machine.

Basic Tools allows users to perform mundane yet necessary functionalities without the need of AI agents, AI programs, cloud storage spaces, proprietary (and often costly) software licenses, internet connection, and user account creation. A tool designed to honor human independence and human actions in the digital space while addressing the lack of attention in the software industry toward creating better user experience.

The tools included in this app are so basic that you can use them in a variety of scenarios, such as professional, research, academic, or personal tasks. When I was learning coding for the first time, I asked the person who opened the world of programming to me the question "Could you please demonstrate what specific real-life situations would we use an accumulator variable?" and got the answer "Everywhere you see." I hope Basic Tools addresses some of the most mundane, basic, yet vexing utilities and user experience gaps that, because of how fundamental they are, affect our digital life "everywhere you see." The goal is for Basic Tools to address these issues.

Project Inception

Scenario 1: I use a MacBook. The MacBook search bar consistently underperforms (especially the search function for a specific directory).

Scenario 2: I take a lot of photos, especially when going on archival research trips. I need a way to easily convert them to different formats and name the photo files in specific ways. There does not exist an efficient workflow that doesn't require expensive Adobe subscription or AI agent.
